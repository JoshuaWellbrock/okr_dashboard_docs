OKR DASHBOARD

OKR Dashboard - EPRO (WS 22/23)

PROJECT STRUCTURE

  • web/assets => contains all assets needed in the application
  • lib => Mainfolder for all the dart code
  • lib/color => contains a CustomColor class to provide a specific color scheme in the application
  • lib/models => contains all data models for the application
  • lib/pages => contains all the pages
  • lib/service => contains a HTTPService class which handles all the HTTP Requests
  • lib/utils => contains utilities like global styling for text and global BorderStyles for TextFields
  • lib/widgets => contains all custom widgets (e.g. Pop-Ups)
  • lib/main.dart => root of the application
